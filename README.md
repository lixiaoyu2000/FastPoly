## Quick Overview
We propose Fast-Poly, a fast and effective 3D MOT method based on the Tracking-By-Detection framework. Building upon our previous work Poly-MOT, Fast-Poly integrates three core principles (Alignment, Densification, and Parallelization) to solve the real-time dilemma of filter-based methods while improving accuracy. **Fast-Poly achieves new state-of-the-art performance with 75.8\% AMOTA among all methods and can run at 34.2 FPS on the nuScenes test leaderboard.**
